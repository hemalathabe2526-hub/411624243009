# Full Stack Placement — Day 1 Project

This project contains the Day 1 HTML and CSS exercise for full-stack placement training.

## Project Structure

- `Day1/Basic.html` — a sample webpage demonstrating HTML structure with headings, paragraphs, images, navigation, lists, tables, forms, video, and grid sections.
- `Day1/Styles.css` — the stylesheet used to style the `Basic.html` page, including typography, layout, buttons, tables, and forms.
- `Day1/` also contains image assets used by the example page.

## Preview My Work
     https://hemalathabe2526-hub.github.io/411624243009/FULL STACK Placement/Day1/Basic.html

## Features

- Basic typography styling for paragraphs and headings
- Circular image styling using `border-radius`
- Responsive navigation layout with `flexbox`
- Custom ordered and unordered list styles
- Styled table rows and cells
- Button hover effects
- Form container styling with rounded corners
- Grid layout examples using `display: grid`

## Notes

- Some CSS rules are commented out for experimentation and learning.
- The project is intended for practicing fundamental HTML and CSS concepts.

## Usage

1. Open `Day1/Basic.html` in a browser.
2. Ensure `Day1/Styles.css` is linked in the HTML file.
3. Review the HTML and CSS to understand how selectors, layout, and styling work together.
