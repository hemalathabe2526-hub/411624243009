# Task4 Theme Switcher

React theme switcher built with `useState`, the Observer Pattern, and Atomic Design.

## Structure

- `src/services/ThemeService.js`
- `src/components/atoms/Button.js`
- `src/components/atoms/Text.js`
- `src/components/molecules/ThemeLabel.js`
- `src/components/organisms/Header.js`
- `src/components/organisms/ThemePanel.js`
- `src/components/organisms/Content.js`
- `src/components/organisms/Footer.js`
- `src/components/templates/MainLayout.js`
- `src/components/pages/HomePage.js`

## Run

```bash
npm install
npm start
```
